## Key decisions

1. **File-copy merge** for codex-co-evolution (zero commits existed; no history to subtree-merge)
2. **Karpathy autoresearch excluded** — unrelated ML training domain; kept as peer project
3. **Evals are the iteration mechanism** (not Karpathy-style auto-research) — upstream 11 pilot bounces missed 8 bugs + 1 scorer blindness that evals caught
4. **`pwsh` optional** — Bash runner does not depend on PowerShell; Bash port of eval harness deferred (~2 days estimated)
5. **Writable-phase default = `false`** — safer posture: Claude refuses to write rather than silently writing garbage if flag is forgotten
6. **Protocol Evolution Loop deferred** — the automated bounce-to-improve-the-bouncer system is future work; requires case library to mature first
